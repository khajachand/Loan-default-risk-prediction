# Loan-default-risk-prediction
This project uses Machine Learning algorithms to predict whether a borrower is likely to default on a loan. The model is trained on financial and personal attributes such as income, loan amount, and credit history to classify applicants as high-risk or low-risk. 

Technologies Used-
### Backend
- Python
- Flask
- Flask-SQLAlchemy
- Flask-CORS
### Machine Learning
- Scikit-learn
- Pandas
- NumPy
- Joblib
### Frontend
- HTML
- CSS
- JavaScript
